# NebulaOS

🌌 Customize Your Universe.

NebulaOS is an Arch Linux–based distribution focused on
customization, simplicity, and a polished space-inspired experience.

## Goals

- Easy to customize
- Beautiful out of the box
- KDE Plasma desktop
- Mission Control settings app
- Built-in widget ecosystem
- Fast and lightweight
